ℹ️ About This Project

The Multi-Agent Book Store is a simulation platform that models interactions between customers, employees, and books in a bookstore environment.

It combines:

  • Multi-agent simulation with Mesa
  • Ontology-based reasoning with Owlready2 and SWRL rules
  • A Flask + WebSocket dashboard for real-time monitoring
  • Sri Lankan localization for books, pricing, and scenarios

Use the dashboard pages to configure simulations, monitor metrics live, review messages and rule execution, and analyze summary outcomes.
